# self-judge

### 项目介绍

这是一个基于 Vue3 + Spirng Boot + Docker 的代码判题系统,由代码沙箱对代码进行处理

前端由管理员创建题目、管理题目，用户可以搜索题目、阅读题目，并对自己感兴趣的题目进行答题

后端用户根据管理员创建的题目进行答题，其中的代码运行在代码沙箱中对代码进行编译、运行及判断答案是否正确

### 技术选型

#### 前端

- Vue3
- Vue-CLI脚手架
- Vuex状态管理
- Arco Design 组件库
- ESLint + Prettier + TS 前端工程化
- Open API 前端API生成

#### 后端

- Spring Boot 
- MySQL
- MyBatis Plus
